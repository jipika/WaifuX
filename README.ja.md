## 🎯 これは何ですか？

**WaifuX** は macOS 用の壁紙アプリです——ただの普通の壁紙ツールではありません。

想像してみてください：朝、Wallhaven の 4K 風景写真でデスクトップを飾り；午後は MotionBGs の動的な流水に切り替え；夜はアニメを見る前に二次元壁紙に変更... そう、私たちは全てを手に入れたい！

> "あなたのデスクトップは、もっと素敵な服を着る価値があります 👔"

---

## ✨ 主な特徴

| 機能 | 説明 |
|------|------|
| 📚 **デュアルライブラリ対応** | Wallhaven（静的）+ MotionBGs（動的）、一つのアプリで二通りの体験 |
| 🎬 **アニメ解析** | ビルトインのマルチソースアニメ動画解析、アニメファンのデスクトップも二次元に |
| 🔍 **スマート検索** | キーワード、タグ、カテゴリ、簡単にお気に入りの壁紙を見つける |
| ⭐ **コレクション管理** | 気に入った壁紙をワンクリックで保存、あなただけの専属ギャラリーを作成 |
| ⚡️ **ワンクリック設定** | 気に入ったものを見つけた？クリック一回で、デスクトップが変わる |
| 🔄 **自動同期** | GitHub ルールの自動更新、ソースサイトが変わっても自動的に適応 |
| 🖼 **フル解像度** | 4K、8K、超ワイド？全て対応 |
| 🌙 **ダークインターフェース** | 眼精疲労防止モード、夜の壁紙変更もまぶしくない |

## 🔧 ルール設定

WaifuX のルールは**動的にロード**されます——つまり、ソースサイトが再デザインされても、私たちはすぐに適応することができます。

ルールは別のリポジトリにホストされています：**[WaifuX-Profiles](https://github.com/jipika/WaifuX-Profiles)**

- アプリ起動時に最新のルールを自動的に同期
- カスタムルールの手動インポートに対応
- ルール形式がシンプルで、自分でも作成可能

```
ユーザーが GitHub URL を入力 → ルールを自動同期 → 使用開始
         ↑___________________________________|
              （ルール更新、自動取得）
```

## ⚠️ 免責事項

WaifuX は単なる**壁紙集約ツール**であり、それ自体では壁紙コンテンツを保存していません：

- [Wallhaven](https://wallhaven.cc) の壁紙は公式 API を通じて取得
- [MotionBGs](https://motionbgs.com) のコンテンツはユーザー自身が設定
- すべての壁紙の著作権は元のウェブサイトと作者に帰属
- 各ウェブサイトの利用規約に従い、個人使用のみに限る
