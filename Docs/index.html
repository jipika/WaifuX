WallHaven

macOS 壁纸应用，支持 WallHaven 壁纸、MotionBGs 动态背景和动漫视频解析

macOS 10.15+ 开源免费 GitHub

主要功能

🖼️

WallHaven 壁纸

浏览和下载 WallHaven 上数百万张高质量壁纸，支持分类筛选、收藏和自动更换

🎬

动态背景

支持 MotionBGs 动态背景，为桌面添加生动有趣的动态元素

📺

动漫解析

集成多个动漫源，支持搜索、播放和下载动漫视频内容

下载最新版

支持 macOS 10.15 (Catalina) 及以上版本

独立的规则仓库

壁纸和动漫解析规则托管在独立的 GitHub 仓库中，支持社区共享和自动同步
